Transition Specifications

All transitions use 200ms cubic-bezier(0.4, 0, 0.2, 1) — Material Design's standard easing curve. Starts fast, decelerates smoothly. No entrance animations, no delays.

Element Property Trigger Effect
Gradient buttons background-position Hover Gradient inverts + smooth 1px lift with shadow. Uses will-change: transform for GPU acceleration.
Outline buttons background-color Hover Subtle background tint (4% black overlay)
List items background-color Hover Subtle background tint (4% black overlay)
Tabs color, background-color Hover Background tint on inactive tabs
Slider thumb transform, box-shadow Hover (track) Scale up to 1.2x with subtle shadow
Swatches / cards transform, box-shadow Hover 1px lift. Light theme: dark shadow. Dark theme: light glow (rgba(255,255,255,0.12))
Selection border border-color, opacity State change Gradient border fades in smoothly
Theme toggle color, background-color, border-color Toggle All themed properties transition together (applied globally via * { transition: color, background-color, border-color 150ms ease })
